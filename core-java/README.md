# Core Java Learning Repository

## Overview

This repository documents a structured and progressive journey through Core Java fundamentals.

The objective is to build a strong foundation in:

- Programming fundamentals
- Object-Oriented Programming
- Interface-driven design
- Core Java library usage
- Modern Date-Time API
- Regular Expressions
- Clean repository organization

The structure follows a **week-wise progression model**, where each week focuses on a specific set of concepts and increases in complexity.

---

## Repository Structure
core-java
├── week-01-core-java
└── week-02-core-java (in progress)


Each week contains multiple assignments covering specific topics.

# Week 01 – Core Java Foundations

Week 01 establishes strong fundamentals in syntax, logic building, object-oriented design, and core Java APIs.

### Topics Covered

- Control Flow & Loops
- Arrays & String Manipulation
- Business Rule Modeling
- Object-Oriented Programming
- Abstract Classes, static & final
- Interface-Based Design
- StringBuilder & StringBuffer
- Wrapper Classes
- Java Math Utilities
- Date Time API
- Regular Expressions
- Formatting & Parsing

### Learning Focus

- Writing clean and modular methods
- Understanding immutability vs mutability
- Applying encapsulation and abstraction
- Designing loosely coupled systems
- Benchmarking performance differences
- Working with real-world validation logic
