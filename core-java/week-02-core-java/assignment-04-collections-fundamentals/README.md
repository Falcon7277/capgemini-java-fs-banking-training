# Assignment 04 – Java Collections Fundamentals

## Overview
This assignment focuses on core Java Collection Framework concepts using real-world scenarios.

The problems demonstrate how different data structures like List, Set, Queue, and PriorityQueue can be used to manage data efficiently.

---

## Concepts Applied
- List implementations (ArrayList, LinkedList)
- Set implementations (HashSet, LinkedHashSet, TreeSet)
- Queue and PriorityQueue
- Duplicate handling and uniqueness
- Order preservation vs sorting
- Custom ordering using Comparator
- Basic data processing using collections

---

## Systems Implemented
1. Student Attendance Manager (Duplicate removal with order preservation)
2. Online Exam Waiting Queue System
3. Email Subscription System (Unique entries)
4. Shopping Cart Application (Dynamic list operations)
5. Hospital Token System (Priority-based processing)
6. Duplicate Removal and Sorting Utility
7. Queue Operations Simulator
8. First Non-Repeated Character Finder
9. Unique Word Counter
10. Ticket Counter Simulation

---

## Learning Outcome
- Selecting appropriate data structures based on use case
- Handling duplicates and maintaining order
- Implementing queue-based systems
- Applying priority-based processing using PriorityQueue
- Using collections to solve real-world problems
