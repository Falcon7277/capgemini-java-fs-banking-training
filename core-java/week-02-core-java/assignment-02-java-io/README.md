# Assignment 02 – Java I/O and Serialization

## Overview
This assignment focuses on Java's Input/Output APIs for file handling, buffered processing, and object serialization.

The problems simulate common real-world scenarios such as file copying, filtering records from files, building a simple task manager, and persisting objects using serialization.

---

## Concepts Applied
- Byte streams (FileInputStream, FileOutputStream)
- Character streams (FileReader, FileWriter)
- Buffered streams (BufferedReader, BufferedWriter)
- File processing using buffers
- try-with-resources for resource management
- Object Serialization and Deserialization
- Serializable interface
- CSV file processing
- List-based data persistence

---

## Systems Implemented
1. File Copy Utility (Byte Stream based file copying)
2. Name Filtering System (File processing using character streams)
3. Console To-Do List Manager with file persistence
4. Student Object Serialization System
5. Employee CSV Processing and Serialization System

---

## Learning Outcome
- Understanding the difference between byte streams and character streams
- Performing safe file operations using try-with-resources
- Implementing buffered reading and writing for efficiency
- Persisting Java objects using serialization
- Processing structured data from CSV files
- Designing simple file-based data storage systems
