# Assignment 03 – Java NIO and High-Performance File Systems

## Overview
This assignment focuses on Java NIO (New I/O) APIs for high-performance file handling and system-level operations.

The problems simulate real-world utilities such as large file processing, directory monitoring, log analysis, and secure backup systems.

---

## Concepts Applied
- FileChannel and ByteBuffer
- Direct vs Heap Buffers
- MappedByteBuffer (Memory-Mapped Files)
- Path and Files API (NIO.2)
- WatchService for directory monitoring
- High-performance file processing
- File system traversal using Files.walk()
- Conditional file copying and backup strategies

---

## Systems Implemented
1. Large File Copy Tool (IO vs NIO performance comparison)
2. Directory Watcher System (Real-time file system monitoring)
3. Log File Analyzer (Memory-mapped file processing)
4. Secure Backup Tool (Incremental file backup system)

---

## Learning Outcome
- Understanding high-performance file handling using NIO
- Comparing traditional IO with NIO approaches
- Processing large files efficiently
- Monitoring file system events in real time
- Designing scalable file-based utilities
