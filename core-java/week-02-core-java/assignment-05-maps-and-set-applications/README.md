# Assignment 05 – Maps and Set Applications

## Overview
This assignment focuses on Java Map and Set implementations using real-world data management scenarios.

The problems demonstrate how different collection types can be used for searching, sorting, uniqueness handling, frequency counting, and maintaining insertion order.

---

## Concepts Applied
- HashMap
- TreeMap
- LinkedHashMap
- HashSet
- Key-Value based data storage
- Automatic sorting using TreeMap
- Insertion order preservation
- Duplicate prevention using Set
- Frequency counting techniques

---

## Systems Implemented
1. Student Marks Lookup System
2. Bank Account Directory
3. Employee Sorting System
4. City Temperature Reporting System
5. Order History Tracker
6. System Log Tracker
7. Online Voting System
8. Unique Number Counter
9. College Admission System
10. Word Frequency Counter

---

## Learning Outcome
- Choosing appropriate Map and Set implementations
- Managing structured key-value data efficiently
- Preventing duplicate entries using Set
- Preserving insertion order using LinkedHashMap
- Implementing sorting and lookup operations
- Solving real-world data management problems using collections
