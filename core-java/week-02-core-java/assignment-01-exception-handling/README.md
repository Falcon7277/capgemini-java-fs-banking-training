# Assignment 01 – Exception Handling

## Overview
This assignment focuses on building robust and fault-tolerant Java applications using structured exception handling.

The problems simulate real-world failure scenarios such as invalid user input, file access errors, business rule violations, and resource management.

---

## Concepts Applied
- Checked vs Unchecked Exceptions
- Custom Exception Classes
- Exception Propagation (`throws`)
- Multi-catch blocks
- Nested try-catch
- Retry mechanisms using loops
- try-with-resources
- Business rule validation
- Regex-based input validation

---

## Systems Implemented
1. Division Calculator
2. File Reader with Retry Mechanism
3. Age Validation System
4. Safe Array Access Utility
5. User Authentication Validator
6. String-to-Integer Conversion Handler
7. File Writing with Resource Management
8. Banking Withdrawal Validation System
9. Nested Exception Demonstration
10. Email Registration Validator

---

## Learning Outcome
- Designing safe and resilient applications
- Creating domain-specific custom exceptions
- Enforcing business rules through exception handling
- Preventing application crashes due to invalid input
- Managing resources safely in file operations
