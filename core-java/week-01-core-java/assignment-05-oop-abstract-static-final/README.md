# Assignment 05 – OOP with Abstraction, static and final

## Overview
This assignment focuses on advanced Object-Oriented Programming concepts
including abstraction, static members, and the final keyword.

Each problem models a real-world system using clean class hierarchy design.

---

## Concepts Applied
- Abstract classes
- Method overriding
- static variables for shared data
- final variables for immutability
- Encapsulation
- Responsibility segregation

---

## Systems Implemented
1. Payment Gateway System
2. Bank Interest Calculation Engine
3. Ride Booking Platform
4. Employee Attendance System
5. Online Examination System
6. E-Commerce Discount Engine
7. Vehicle Toll Management
8. Library Management System
9. Online Food Delivery Platform
10. Insurance Premium Calculator

---

## Learning Outcome
This assignment strengthens:
- Class hierarchy design
- Shared state management using static
- Immutable design using final
- Clean extensible architecture
- Scenario-based abstraction modelling
