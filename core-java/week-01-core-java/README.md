# Week 01 – Core Java Foundations

## Overview

This week focuses on building a strong foundation in Core Java.
The assignments progressively cover control flow, arrays, OOP principles,
interfaces, core library classes, Date Time API, and Regular Expressions.

The objective is to move from basic programming constructs to
structured object-oriented and interface-driven design.

---

## Assignments Covered

### Assignment 01 – Control Flow & Arrays
- Conditional logic
- Loop constructs
- Array traversal
- Basic problem solving

### Assignment 02 – Decision-Making Systems
- Rule-based validation systems
- Multi-condition evaluation
- Business logic modelling

### Assignment 03 – Arrays & Strings
- Array manipulation
- String processing
- Frequency analysis
- Word-based operations

### Assignment 04 – OOP Banking System
- Encapsulation
- Class design
- Method-based business logic
- Responsibility separation

### Assignment 05 – Abstraction, static & final
- Abstract classes
- Method overriding
- Static shared state
- Final immutability concepts

### Assignment 06 – Interface Driven Design
- Interface-based architecture
- Runtime polymorphism
- Default and static interface methods
- Loose coupling principles

### Assignment 07 – String, Math & Wrapper Classes
- Immutable vs mutable strings
- Performance benchmarking
- Math utilities
- Wrapper class behavior

### Assignment 08 – Date Time API & Regular Expressions
- Modern Date Time API usage
- Regex-based validation
- Currency formatting
- Text parsing and extraction
