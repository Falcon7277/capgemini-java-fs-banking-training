# Assignment 02 – Decision Making & Business Logic Systems

## Overview
This assignment models real-world business logic scenarios using Java.
Each problem implements multi-condition validation and rule-based decision systems.

All solutions are written as independent methods inside `Main.java`.

---

## Concepts Covered

### Rule-Based Systems
- Loan approval validation
- University admission eligibility
- Insurance policy validation
- ATM withdrawal checks

### Financial Calculations
- Income tax computation
- Electricity billing with slabs
- Cab fare estimation
- Employee bonus calculation

### Conditional Evaluation
- Nested if-else logic
- Multi-parameter validation
- Boolean condition handling

---

## Key Techniques Used
- Structured conditional branching
- Business rule modelling
- Input normalization (trim, lowercase)
- Multi-condition evaluation

---

## Learning Outcome
This assignment improves:
- Logical reasoning
- Designing validation systems
- Translating real-world rules into code
- Clean and modular method implementation
