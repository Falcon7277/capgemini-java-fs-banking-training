# Assignment 06 – Interface Driven Design

## Overview
This assignment focuses on industry-oriented interface-based architecture.
It emphasizes loose coupling, runtime polymorphism, and clean extensible design.

---

## Concepts Applied
- Interface-based programming
- Runtime polymorphism
- Strategy pattern fundamentals
- default methods in interfaces
- static helper methods in interfaces
- Loose coupling
- Swappable implementations

---

## Systems Implemented
1. Enterprise Notification Dispatch System
2. Pluggable Authentication Framework
3. Multi-Vendor Cloud Storage System
4. Online Order Pricing Engine
5. Enterprise Report Generation Engine

---

## Learning Outcome
- Designing systems using contracts (interfaces)
- Eliminating conditional logic using polymorphism
- Building extensible architectures
- Applying industry design principles
