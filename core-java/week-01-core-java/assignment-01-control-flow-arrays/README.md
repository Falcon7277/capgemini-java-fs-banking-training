# Assignment 01 – Control Flow & Arrays

## Overview
This assignment focuses on strengthening core Java fundamentals using
control flow statements and array-based problem solving.

All problems are implemented using method-based structure inside `Main.java`.

---

## Concepts Covered

### Control Flow
- Conditional statements (if-else)
- Nested condition evaluation
- Loop constructs

### Arrays
- Counting odd and even elements
- Prime number generation
- Fibonacci sequence
- Salary analysis
- Exam result analysis
- Inventory stock checking

### Logical Problem Solving
- Factorial computation with constraints
- Digit reduction to single digit
- Pattern printing

---

## Key Techniques Used
- Iterative loops
- Conditional branching
- Basic mathematical logic
- Defensive input validation

---

## Learning Outcome
This assignment builds a strong foundation in:
- Flow control
- Array traversal
- Writing modular methods
- Handling edge cases in simple programs

---

## File Structure
- `Main.java` → Contains all solutions
- `assignment-01-control-flow-arrays-questions.pdf` → Problem statements
