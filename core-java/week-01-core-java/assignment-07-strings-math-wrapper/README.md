# Assignment 07 – String, StringBuffer, StringBuilder, Math & Wrapper Classes

## Overview
This assignment focuses on core Java library classes including
String manipulation, mutable string handling, mathematical utilities,
and wrapper class behavior.

The objective is to understand the practical differences between
immutable and mutable strings, performance characteristics,
and numeric utility methods provided by the Java standard library.

---

## Concepts Covered

### 1. String Class
- Finding length of a string
- Extracting substrings
- Converting case (upper/lower)
- Palindrome checking
- Anagram detection
- Character frequency calculation
- Word reversal

### 2. StringBuffer
- Append, insert, replace operations
- Reverse operation
- Performance comparison with String
- Removing vowels from a string

### 3. StringBuilder
- Efficient string modification
- Word counting and appending metadata
- Performance benchmarking (10,000 concatenations)

### 4. Math Class
- Power calculation (x^y)
- Square root
- Absolute difference
- Random number generation
- Circle area and circumference calculation
- Rounding functions (ceil, floor, round)
- Trigonometric functions (0° to 360°)

### 5. Wrapper Classes
- Boxing and unboxing
- Auto-boxing and auto-unboxing
- String-to-number data conversion
- Primitive vs Wrapper comparison using `==` and `.equals()`

---

## Learning Outcomes

- Understand immutability vs mutability in strings
- Analyze performance differences between String, StringBuffer, and StringBuilder
- Use built-in mathematical utilities effectively
- Understand wrapper class behavior and object comparison
- Strengthen familiarity with core Java standard library classes
