# Assignment 03 – Arrays and Strings

## Overview
This assignment focuses on fundamental array and string manipulation problems in Java.  
It covers searching, traversal, rotation, matrix processing, and basic string analysis.

All solutions are implemented inside `Main.java`.

---

## Concepts Covered

### Arrays
- Finding maximum and minimum elements
- Left rotation of array (in-place reversal method)
- Counting even and odd elements
- Finding second largest element
- Row-wise sum of a matrix

### Strings
- Palindrome check (two-pointer technique)
- Counting vowels
- Character frequency calculation
- Finding the longest word in a sentence
- Counting names starting with a given character

---

## Key Techniques Used
- Loop traversal
- In-place array reversal
- Two-pointer approach
- String splitting using regex
- Character manipulation using `Character` class

---

## Learning Outcome
This assignment strengthens:
- Logical problem solving
- Array indexing and boundary handling
- String processing techniques
- Clean method-based implementation structure
